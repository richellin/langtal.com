# langtal.com
This is [langtal](https://langtal.com)

### Use
+ PHP 7
+ Laravel5.2
+ Elixir

### Setup
```sh
#Clone the repository
git clone https://github.com/richellin/langtal.com.git

#Laravel Setting
composer install
php artisan key:generate

#Install package.json 
npm install

#Elixir Running Task
npm run prod

#Start artisan
php artisan serve 
#OR
php artisan serve --host 0.0.0.0

#Open your browser to 
http://localhost:8000
```
